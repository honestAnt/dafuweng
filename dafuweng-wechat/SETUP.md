# 微信小游戏配置指南

## 📋 项目转换完成

您的糖果大富翁项目已成功从微信小程序转换为微信小游戏！

## 🎯 主要变化

### 文件结构变化
**之前（小程序）：**
```
dafuweng-wechat/
├── app.js
├── app.json
├── app.wxss
├── pages/
│   ├── index/
│   └── game/
└── ...
```

**现在（小游戏）：**
```
dafuweng-wechat/
├── game.js          # 主游戏逻辑
├── game.json        # 游戏配置
└── ...
```

### 技术变化
- ✅ 渲染方式：从 WXML/WXSS 改为 Canvas 2D
- ✅ 交互方式：从组件事件改为触摸事件
- ✅ 配置文件：使用 game.json 替代 app.json
- ✅ 项目类型：compileType 设置为 "game"

## 🚀 在微信开发者工具中配置

### 1. 打开项目
1. 打开微信开发者工具
2. 点击 "+" 号导入项目
3. **重要**：选择项目类型为 **"小游戏项目"**
4. 选择目录：`/Users/admin/Documents/code/github/dafuweng/dafuweng-wechat`
5. 填写 AppID（可以使用测试号）
6. 点击"导入"

### 2. 验证配置
导入后，检查以下配置：

**project.config.json 应该包含：**
```json
{
  "compileType": "game",
  "appid": "wxbec90314ae0a5351",
  "projectname": "dafuweng-wechat"
}
```

**game.json 应该包含：**
```json
{
  "deviceOrientation": "portrait",
  "showStatusBar": false
}
```

### 3. 编译运行
- 点击工具栏上的"编译"按钮
- 游戏应该在小游戏模拟器中运行
- 可以使用鼠标或触摸屏进行操作

## 🎮 游戏操作

### 开始界面
- 点击数字按钮选择玩家人数（2-5人）
- 点击角色卡片选择角色
- 点击"开始游戏"按钮进入游戏

### 游戏界面
- 点击"掷骰子"按钮移动角色
- 如果可以购买地产，会显示"购买地产"按钮
- 点击"结束回合"按钮结束当前玩家回合
- 点击棋盘格子可以查看详细信息

## 🔧 常见问题

### Q: 编译时提示 "game.json 未找到"
**A:** 确保在微信开发者工具中选择的是"小游戏项目"而不是"小程序项目"

### Q: 预览时提示 "非法的文件" 或 "invalid file"
**A:** 小游戏不支持 .wxml 和 .wxss 文件，确保项目中只有 game.js 和 game.json

### Q: 游戏无法显示
**A:** 
1. 检查 game.js 文件是否存在
2. 检查 Canvas 是否正确初始化
3. 尝试清除缓存后重新编译

### Q: 触摸事件不响应
**A:** 
1. 确保使用的是小游戏模拟器
2. 检查 wx.onTouchStart 是否正确注册
3. 尝试重启开发者工具

### Q: 如何切换回小程序版本
**A:** 
1. 从 git 历史记录恢复小程序版本
2. 或者重新创建小程序版本
