# 糖果大富翁 - 微信小游戏版

这是糖果大富翁游戏的微信小游戏版本，专为儿童设计，界面友好，玩法简单有趣。

## 📱 功能特性

- ✅ 完整的游戏功能实现
- ✅ 支持2-5人游戏
- ✅ 20种可爱角色可供选择
- ✅ 丰富的游戏元素：地产、陷阱、天气、特殊建筑
- ✅ 儿童友好的界面设计
- ✅ Canvas 渲染，流畅的游戏体验
- ✅ 触摸操作，适配移动设备

## 🚀 快速开始

### 1. 安装微信开发者工具

下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"小游戏项目"
3. 选择 `dafuweng-wechat` 目录
4. 填写 AppID（如果没有，可以选择测试号）
5. 点击"导入"

### 3. 运行项目

导入后，微信开发者工具会自动编译并预览项目。

## 🎮 游戏玩法

### 1. 选择角色
- 选择玩家人数（2-5人）
- 点击选择对应数量的可爱角色
- 点击"开始游戏"进入游戏

### 2. 游戏操作
- 点击"掷骰子"按钮移动
- 到达无主地产时可以购买
- 到达他人地产时需要支付租金
- 完成行动后点击"结束回合"

### 3. 游戏元素
- **地产**：购买后可以收取租金
- **陷阱**：会扣除金币
- **天气**：烈日、风雪、暴雨、彩虹等特殊效果
- **特殊建筑**：银行、监狱、休息区、机会卡

## 🎨 技术栈

- **微信小游戏原生框架**
- **Canvas 2D**：游戏渲染
- **JavaScript**：游戏逻辑
- **触摸事件**：用户交互

## 🔧 配置说明

### project.config.json
- `appid`: 小游戏 AppID
- `projectname`: 项目名称
- `compileType`: 编译类型（game）
- `libVersion`: 基础库版本

### game.json
- `deviceOrientation`: 屏幕方向（portrait 竖屏）
- `showStatusBar`: 是否显示状态栏
- `networkTimeout`: 网络超时设置

## 🔄 从小程序转换为小游戏

本项目已从微信小程序转换为微信小游戏，主要变化：

1. **渲染方式**：从 WXML/WXSS 改为 Canvas 2D
2. **文件结构**：简化为 game.js、game.json（小游戏不需要 wxml 和 wxss 文件）
3. **交互方式**：从组件事件改为触摸事件
4. **配置文件**：使用 game.json 替代 app.json
