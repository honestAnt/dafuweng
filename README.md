# 🍭 糖果大富翁 - 儿童版

一个专为儿童设计的趣味大富翁游戏，采用糖果主题，界面友好，玩法简单有趣。

![糖果大富翁](https://img.shields.io/badge/版本-1.0.0-brightgreen) ![HTML5](https://img.shields.io/badge/技术-HTML5-orange) ![JavaScript](https://img.shields.io/badge/技术-JavaScript-yellow) ![CSS3](https://img.shields.io/badge/技术-CSS3-blue)

## 🎮 游戏特色

- **儿童友好设计**：可爱的动物角色、糖果主题、明亮的色彩
- **简单易玩**：点击掷骰子即可开始，规则简单易懂
- **丰富玩法**：地产购买、陷阱、天气系统、特殊建筑等多种元素
- **多人游戏**：支持2-5人同时游戏
- **自定义角色**：20种可爱角色可供选择
- **响应式设计**：适配不同屏幕尺寸

## 🚀 快速开始

### 方法一：直接打开
1. 下载或克隆本项目
2. 用浏览器打开 `index.html` 文件
3. 开始游戏！

### 方法二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 使用 PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 游戏规则

### 🎯 基本玩法
- 掷骰子移动，购买地产，收取租金
- 金币用完的玩家出局，最后剩下的玩家获胜
- 经过起点获得200金币奖励

### 🏠 地产系统
- 购买地产后可以收取租金
- 租金根据地产价格计算
- 经过自己的地产可以休息

### ⚠️ 陷阱格子
- 粘粘糖、酸柠檬等会扣除金币
- 陷阱惩罚金额不等
- 注意避开危险区域

### 🌤️ 天气系统
- **☀️ 烈日**：停留1回合，失去50金币
- **❄️ 风雪**：后退3步，失去30金币
- **🌧️ 暴雨**：前进2步，获得20金币
- **🌈 彩虹**：随机传送，获得50金币

### 🏢 特殊建筑
- **🏦 银行**：存钱生息，取钱获得利息
- **🏢 监狱**：停留2回合，无法移动
- **🏥 医院**：恢复100金币
- **🎰 赌场**：50%几率双倍或失去金币
- **🎁 礼物屋**：随机获得金币礼物
- **🌀 传送门**：随机传送到其他位置
- **💎 宝藏岛**：发现宝藏获得金币
- **🧙 魔法屋**：随机魔法效果

## 🎯 游戏玩法

1. **选择玩家人数**：点击2-5人游戏按钮
2. **选择角色**：点击选择对应数量的可爱角色
3. **开始游戏**：点击"开始游戏"按钮
4. **掷骰子**：点击掷骰子按钮移动
5. **购买地产**：到达无主地产时可以购买
6. **支付租金**：到达他人地产时需要支付租金
7. **特殊效果**：触发各种天气、陷阱、特殊建筑效果
8. **结束回合**：完成行动后结束回合，轮到下一位玩家
